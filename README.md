# Projeto_Pousada
Projeto simples em Python desenvolvido para a disciplina de Software Básico. O programa simula um sistema de cotação de hospedagem, onde é possível escolher o tipo de apartamento, informar quantidade de hóspedes e dias de estadia. O sistema realiza validações de entrada e calcula automaticamente o valor final da reserva.

Tecnologias utilizadas:
- Python

Conceitos praticados:
- Entrada de dados com input
- Estruturas condicionais (if/else)
- Estruturas de repetição (while)
- Tratamento de erros com try/except
- Uso de dicionários para armazenar valores
- Validação de dados digitados pelo usuário

Objetivo do projeto:
- Praticar conceitos básicos de programação em Python e lógica de programação, desenvolvendo um pequeno sistema de cotação executado diretamente no terminal.
